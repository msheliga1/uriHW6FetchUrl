Mike Sheliga's Testing of Column Margins

Main things seem to be

(1) Use unclosed container - row - col - col SubDiv

(2) Dont put any text, nor a border after/in the col div

(3) container get padding, row get NEGATIVE margin

(4) https://stackoverflow.com/questions/51160147/bootstrap-4-how-to-have-margin-between-columns-without-going-over-space claims that setting MARGINS in the cols "breaks" the bootstrap grid. (padding likely fine)
